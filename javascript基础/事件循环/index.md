## event Loop

`JS`是一门单线程的语言，从名字就知道，在整个`JS`执行的过程中，有且只能存在一个**线程（主线程）**：是操作系统能运算调度的最小单位。它被包含在`进程`当中，是`进程`的实际运作单位。一条线程指的是进程中一个单一顺序的控制流，一个进程内可以包含多个线程，**每个线程并行执行不同的任务**。线程的特点：

- 轻量实体
- 独立调度和分派的基本单位
- 可以并发执行
- 共享线程资源

**进程是资源分配的基本单位**
